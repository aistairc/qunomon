QunomonはAIシステムの品質評価支援ツールです。
機械学習マネジメントガイドライン に沿ったAIシステムの品質評価とその作業環境を提供します。

Download (Free)


機械学習品質ガイドラインを踏襲したテスト管理の共通基盤

本テストベッドは、AIシステム開発者・AIシステム品質評価者・評価手法開発者が共同で参加し、 AIシステムの品質について定量的に評価するとともに、開発プロセスや評価記録・検証など品質管理の包括的な支援を行うことができ、 品質に関する不透明性の解消やビジネス活用の加速が期待できます。



品質要件による機械学習モデルの品質評価に関するレポートの自動生成

AIシステムに必要な機械学習モデルやデータセットの品質をテストするさまざまなプログラムの実行結果から品質レポートを自動生成できます。 AIシステムをガイドラインに準拠した品質観点でチェックし、その結果を第三者に提供できる資料として簡単に作成できます。



豊富なAIシステム評価パッケージ（AIT: AI system Test package）をプレインストール

AIシステム開発に必要なデータセットや機械学習モデルなど、機械学習要素についての具体的な品質尺度を測定する ソフトウエアプログラムをパッケージ化し、評価結果を管理する共通基盤上で アプリケーションプログラミングインターフェース（API）経由でAITを利用することができます。 AIT_SDKを用いることで、AI開発者が様々なテスト資産をAITとして登録することができます。
自社用のテストコードを資産管理したり、第三者の作成したAITを利用することで、AIシステムの品質要件に合った評価環境にカスタマイズできます。

※AITの公開・カスタマイズ機能はα版では未公開となります


柔軟なシステムの拡張性

新しく開発された機械学習モデルや品質測定技術などを柔軟に取り入れる仕組みを構築し、 開発プロセス支援と評価記録・検証とを両立させる一元的な作業環境のオープンソフトウエア群を提供します。
