3.7. Push to GITHUB¶

This chapter describes push GITHUB to publish the verified AIT.

3.7.1. Execute batch of push to GITHUB¶

Execute the batch file and push to GITHUB.

Windows
{YOUR_AIT_ROOT}/tool/github_push.bat

linux
{YOUR_AIT_ROOT}/tool/github_push.sh

3.7.2. Select upload inventory¶

Choose whether to upload inventory for AIT test.

Y: Upload.

N : Do not upload.

3.7.3. Enter commit comment¶

※Since authentication to GITHUB is required.

HTTPS type: token

SSH: Public key
